# Cost Optimization Tool

## Overview

Cloud cost analysis tool that identifies unused resources, recommends right-sizing, and tracks spending trends across AWS services.

## Key Features

- EC2 instance right-sizing recommendations
- Unused EBS volume and snapshot detection
- S3 storage class optimization suggestions
- Cost trend analysis and forecasting
- Exportable CSV/JSON reports

## Concepts Used

- Module 08: Infrastructure as Code (resource inventory)
- Module 14: Data Processing (CSV, JSON reports)
- AWS APIs (boto3)
